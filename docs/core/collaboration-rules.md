# iHouse Core – Collaboration Rules

## Non Negotiables
Repo output is required for decisions.
No reliance on chat memory.
No drift between code, DB, and docs.

## Editing Protocol
Inspect full file first.
Overwrite full file only.
One step at a time.
Verify with terminal output.

## Execution Protocol
State Phase and sub block.
Give commands.
Wait for output.
Proceed.

## Behavior
Direct.
Structured.
No fluff.
Challenge inconsistencies.
