# Domaniqo Brand Handoff

This note is meant to bring any new agent, collaborator, designer, or operator into the project with the current state of the brand.

Please treat this as the current working source of truth unless a newer update is provided.

---

# 1. Important Update: the brand name has changed

The company should no longer be referred to externally as **iHouse**.

That name is no longer the public-facing brand and should not be used in outward brand, product, website, marketing, or presentation work.

The brand is now:

**Domaniqo**

Primary domain:

**domaniqo.com**

Important rule:

**Do not return to iHouse as the external name.**

If needed, iHouse may remain only as a temporary internal codename in legacy references, but it should not appear as the external brand.

# 7. Tone of voice

The tone should be:

calm  
clear  
premium  
restrained  
human  
precise

Use short, direct, confident language.

Prefer measured statements over hype.

The brand should not sound:
salesy  
cute  
startup-loud  
buzzword-heavy  
overwritten  
overly technical in public-facing copy

Good examples of tone:

Run every stay with clarity.  
See every stay.  
Calm command for modern hospitality.  
One place to manage operations, stays, and standards.

Avoid language like:

Revolutionize your property workflow  
The all-in-one super app for hosts  
AI-powered magic for modern rentals  
Next-generation disruption platform

# 13. Typography

Current recommended type system:

## Primary brand type
**Manrope**

Used for:
headlines  
brand UI emphasis  
structured modern communication

Reason:
clean  
modern  
premium-digital  
structured  
warm enough without being decorative

## Secondary / UI type
**Inter**

Used for:
body copy  
product UI  
documents  
tables  
settings  
dashboard text

Reason:
excellent clarity and readability

## Editorial accent only
**Instrument Serif**

Used sparingly for:
hero statements  
campaign headers  
high-end editorial moments

Important:
Instrument Serif is not the main body type and should not dominate the product.
It is an accent, not the whole system.

### Recommended setup
Headlines: Manrope SemiBold  
UI and body: Inter Regular / Medium  
Editorial moments only: Instrument Serif

# 15. Color usage logic

The brand should mostly live in:

dark graphite  
stone  
white  
moss

Copper is a premium accent only.

Do not overuse copper.

Copper should not become the dominant color across the entire system.

### Suggested usage ratio
60% neutrals  
25% darks  
10% greens  
5% copper accents

### Role guidance
Midnight Graphite:
primary dark, main text, navigation, strong contrast

Cloud White:
main background

Stone Mist:
soft premium light surface

Soft Sand:
secondary warm neutral

Deep Moss:
buttons, CTAs, strong interface actions

Quiet Olive:
secondary natural operational tone

Signal Copper:
premium highlight only

Muted Sky:
secondary information tone, light system support, optional restrained accent

9. Produce clean, implementation-ready brand and UI outputs
