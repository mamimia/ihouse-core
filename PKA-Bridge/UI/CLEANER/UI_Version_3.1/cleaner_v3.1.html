Cleaner — Operational UI Direction v3.1

Refined spatial, photo-first worker surface. Room-by-room progression with clarity gates.

V3.1 REFINEMENT

Photo action buttons more prominent. Room tabs feel like spatial navigation. Supply state cycling more tactile. Completion gates emphasize quality checkpoint. Success screen reinforces professional pride. Active clean card dominates home view.

SPATIAL IDENTITY PUSH

Worker always knows which room they're in, what's done, what's next. Room dots and tabs now spatial anchors. Checklist items have left-accent borders when checked. Supply chips instant visual feedback. Progress bars stay visible on all screens.

PHOTO-FIRST INTEGRATION

Photo badge prominent on active card (green accent). Photo action button (📷) is larger, centered, more tappable. Photo progress bar visible on checklist. Supply and issue screens also support photo capture. Central to worker workflow, not bolted on.

COMPLETION GATE CLARITY

3-flag system (rooms cleaned, photos captured, supplies verified) feels like real quality checkpoint. Green flags feel earned. Blocker messages are clear and actionable. Override button feels intentional. Success animation and stats reinforce completion.
