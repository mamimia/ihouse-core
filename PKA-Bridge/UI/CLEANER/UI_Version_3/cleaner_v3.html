6-screen wizard. 3-flag completion gate. Photo-first, room-driven worker surface.

ROLE PERSONALITY

Spatial, room-driven, photo-first. Cleaner sees property name, room count, checklist, supply par levels, photo documentation per room, and 5-min SLA critical issues.

V3 CHANGES FROM V2

Expanded from 3 to 6 screens. Added real 3-flag completion gate with blocker messaging. Supply check as distinct screen with 4-state cycle. Issue reporting with critical SLA warning. Aligned to 21 items / 5 rooms / 7 supplies.

KEY PATTERNS

3 progress bars always visible. Room tabs on checklist. Photo badges + inline 📷 buttons. Supply 4-state cycle (unchecked → ok → low → empty). 3-flag gate with blocker explanation. Ready-pulse animation.
