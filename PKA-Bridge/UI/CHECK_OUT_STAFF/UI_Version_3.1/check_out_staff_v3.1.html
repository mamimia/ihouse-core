Operational UI Direction — Version 3.1 (Refinement)

V3.1 Refinement Focus Closure strengthened: departure cards now have dominant property name (13px/800), stronger status badging, and arrival time displays clearly. Day health strip redesigned with copper border accent to show progress visually. Summary section moved up with dark background for authority. Early departure banner now iconographically prominent. All screen step headers darkened with copper border for field-tool authority.
Verification Tightening Photo comparison tabs now feel professional—active tab has copper background with subtle shadow. Room selector tabs redesigned as actual document-like tabs (darker active state). Photo area guidance clearer. Property Inspection step displays room name prominently in dark header. Damage/Issue reporting now has uppercase CRITICAL labels for severity pills (low/medium/high/critical). Settlement cards enlarged to 20px amounts for unambiguous reading.
Turnover Confidence + Settlement Authority Settlement summary redesigned as financial document: breakdown rows clearer, net return amount now 13px/800 in copper. Delta preview tightened with dark background, copper border accent, larger charge value (13px/800). Success screen has gradient header (copper to darker copper) for conclusiveness. Turnover chain step icons now bolder (800 weight). Signature/completion warning more visually urgent. All monetary values now 800-weight Manrope for authority.
Visual Polish + Field Tool Language Removed "board" feeling: removed floating toggles, made buttons more field-like (12px padding, rounded 8px). Copper accent (#B56E45) consistently used for active states, progress, and authority. All form labels now 700-weight uppercase with 0.5px letter-spacing. Radio options and settlement cards have 800-weight labels. Bottom nav button text now 700-weight. Shadows refined (0 1px 3px) to feel grounded, not floating.
