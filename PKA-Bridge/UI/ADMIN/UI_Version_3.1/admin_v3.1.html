V3.1 Alignment Pass
VISUAL ALIGNMENT ONLY. This pass refines Admin's visual language to match V3.1 system design standards: Manrope 800 weight headers, tightened typography, refined card shadows, dark bottom nav, left-accent 3px borders, uppercase screen labels with letter-spacing.
Admin Desktop-Canonical Question
Admin is the SYSTEM MANAGEMENT surface — user management, property configuration, system settings, permissions. The deeper question of whether Admin should be desktop-canonical (not phone-first) is DEFERRED to V4. All V3 screens/behavior preserved.
What Changed from V3
Screen labels now 11px, 700 weight, uppercase, #6B7280, 0.5px letter-spacing. Header font weights increased to 800. Bottom nav darkened to #171A1F, height 48px. Card left-accent borders 3px. Badge weights to 800. Spacing tightened 1-2px. Dark header border refined to rgba(255,255,255,0.08).
Preserved from V3
All 6 screens intact: System Overview, People Management, Property Governance, Intake Review, Manager Delegation, Integrations & Audit. All functionality, interactions, cards, patterns unchanged. Phone mockups 300x622px, status bar 26px. This pass is visual refinement only.
