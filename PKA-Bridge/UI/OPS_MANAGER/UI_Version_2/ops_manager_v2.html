Role Personality
Command center. Oversight. The Ops Manager sees everything happening across the operation — they coordinate, intervene, and dispatch. Not a worker; a LEADER.
Design Anchor
This IS the anchor. All other surfaces derive visual direction from Ops Manager. It sets the tone for urgency coding, intervention patterns, and command-line aesthetics.
Key Patterns
KPI strips (dark). Stream cards. Alert urgency coding. Intervention buttons. Cockpit analytics. Information-dense but never overwhelming. Always showing what needs attention NOW.
Shaped By
Larry (cross-domain coordination). Talia (interaction architecture). Hana (staff operations). A military-grade dashboard compressed into a phone.
