What Was Refined
Screen 2 is the active step flow — the execution wizard. Type separation is strong: Deep Moss = arrivals, Signal Copper = departures, carried through card borders, badges, and active flow header.
Urgency System
4 states on every task card. Safe → Approaching (amber, >30 min) → Imminent (copper, <30 min, pulsing) → Overdue (red, blinking). Day Health strip shows live stream counts.
Card Scan Order
Worker reads left to right: type badge (ARRIVAL/DEPARTURE) → countdown pill → nav icon → property name → guest + time → CTA. All critical info in under 2 seconds.
What Stayed Calm
History and Profile have no urgency animation. They are reporting surfaces, not action surfaces. The active flow screen uses copper only for the current-step indicator.
