V3.1 Alignment
Screen labels: 11px Manrope 700 #6B7280 with 0.5px letter-spacing. Notes bar on dark #171A1F background in 4-column grid. Font weights increased to 800 in headers, badges, key titles. Spacing tightened by 1-2px throughout. Dark header borders softened to rgba(255,255,255,0.08).
Mixed-Role Coherence
This surface combines check-in AND check-out into one coherent tool for workers handling both. Home screen shows both arrivals and departures in unified queue, clearly distinguished but not segregated. Check-in flows align with CHECK_IN_STAFF V3.1. Check-out flows parallel CHECK_OUT_STAFF patterns. One shell, two capabilities.
Visual Refinements
Card shadows refined (subtle, grounded: 0 1px 2px). Left-accent 3px borders for all urgent/typed elements. Phone: 300x300x622px, border-radius 36px. Status bar: white background. Bottom nav: 56px height. Task type (arrival/departure) as clear first indicator, property dominant, guest and time follow hierarchy.
Typography & Colors
Display/labels: Manrope 800. Body: Inter 400-500. Colors: deep-moss #334036, signal-copper #B56E45, midnight #171A1F. Badge badge-ready #D4E4D9/#334036 (moss), #F5EAE3/#B56E45 (copper). No new behavior invented — only visual/product alignment from V3.
